# mux Scaffold
`mux Scaffold` is CLI to generate scaffolds using `gorilla/mux` router/dispatcher.

## Installation

``` shell
go get github.com/nasa9084/mux-scaffold
```

### Initializing a project

``` shell
mux-scaffold init <project path>
```
